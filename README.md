# 맛있다(Masitda)

<br>

**맛있다 (Masitda)** 는 다양한 식품을 구매할 수 있는 반응형 쇼핑몰 웹페이지입니다. 비회원 구매하기를 지원하여, 회원과 비회원 모두 이용할 수 있는 쇼핑몰입니다.
프론트엔드는 React.js 라이브러리, 백엔드는 express.js 프레임워크로 구현되어 있습니다.
각각의 상세 내용은 하단의 링크를 통해 확인할 수 있습니다.
